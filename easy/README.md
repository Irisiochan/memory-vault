# Windows 开箱版

请从 GitHub Releases 下载 `Memory-Vault-Windows-x64.zip`，不要下载 GitHub 自动生成的 Source code 压缩包。

1. 解压到一个普通文件夹（不要直接在 ZIP 里运行）。
2. 双击根目录 `start.cmd`。
3. 浏览器自动打开后，点默认联系人设置。
4. 填写 API Base URL、模型名和 API Key，保存后即可聊天。

便携版内置 Python、Node.js 和 Memory Vault MCP，默认仅监听本机地址。聊天、Key 和记忆都保存在解压目录中；升级前请备份该目录。

需要 Claude CLI、Codex、Tailscale、VPS 或源码调试时，请使用 `app/README.md` 中的开发者版。
